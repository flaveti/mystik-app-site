# Design Preferences for Mystik Project

## Layout Preferences

### ❌ Avoid
- **Emojis in layouts**: The user prefers clean layouts without decorative emojis
- Keep interfaces minimal and professional
- Focus on typography and spacing over decorative elements

### ✅ Prefer
- Clean, modern design
- Proper spacing and typography
- Subtle visual elements (dots, lines, geometric shapes)
- Gradient backgrounds and effects
- Icon-based navigation and indicators

## Notes
- Functional emojis (like country flags for language selection) are acceptable
- Focus on mystical theme through colors, gradients, and animations rather than symbols
- Maintain professional spiritual aesthetic
